Quels genres de films parmi les plus populaires remportent le plus de récompenses (awards) ?

Quel graphe de connaissances allez-vous interroger ?

La requête SPARQL afin de répondre à la question

    SELECT ?nomGenre (COUNT(?genre) AS ?nbAwardsReçus)
    WHERE {
      ?genre a iut:Genre;
             iut:nom ?nomGenre.
      ?film a iut:Film;
            iut:nom ?nomFilm;
            iut:aPourGenre ?genre;
            iut:award ?medailleAward.
    }
    GROUP BY ?genre ?nomGenre
    ORDER BY DESC(?nbAwardsReçus)
    LIMIT 100
